Frequently Asked Questions (FAQ)

What is the Document Object Model (DOM)?

Browser creates a Objects model when a web page is loaded. It is constructed as a tree of elements and nodes. Using JavaScript we can modify the html elements by selecting them using different JavaScript DOM methods.

How do you select an element from the DOM?

There are various methods for select an element from the DOM. It can select by ID,Class Name,Query Selector and Tag Name

What is event delegation in the context of the DOM, and why is it useful?

It can bubbling event propagation to handle event. It is useful for manage and handle events on multiple child elements efficiently

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes by getAttributes, setAttributes or removeAttributes and style property using DOM for style an element. Also add or remove classes value using classList and className
